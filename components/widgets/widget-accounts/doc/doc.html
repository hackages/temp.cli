Accounts Widget

Introduction



The Launchpad Accounts widget displays a list of accounts tied to the user currently logged in. Built using AngularJS, the Accounts widget displays, at the time of writing, the following informations about the user's accounts:

  • Account list groups — the accounts are split up in these 3 default groups:
    1. Deposit & Saving
    2. Cards & Debt
    3. Investment
  • The name of the account (ex: Personal Checking Account)
  • The International Bank Account Number (IBAN)
  • The current or available balance (currency is specified by the service)
Further to the above, selecting an account will open the Transactions widget, displaying a detailed list of the accounts transactions.

Functionality

The Accounts widget provides all the details needed for a quick look at your accounts, including the account name, the account IBAN, and the current/available account balance.

Accessibility

The Accounts Widget is fully accessible via keyboard control, as well as with a screen reader. The account can be selected with either Enter or Space key.

Configuration

The following table describes the different preferences that can be configured for the accounts widget

Preference Description
groupsDataSrc The URL to retrieve the account list groups
showGroups A flag to specify if the account groups are displayed
accountsDataSrc The URL to retrieve the list of the users accounts
defaultBalance A flag to specify whether the widget should display the current or the available balance

Pubsub

The below table details the different pubsub messages that the widget publishes/subscribes to:

Event Type Arguments Description
launchpad-retail.accountSelected Publish {accountId: id, originType: "accounts"} This message is published to any interested parties when a new account is selected

Dependencies

The following Launchpad Modules are dependencies of the Accounts Widget:

  • Launchpad UI
  • Launchpad Accounts
  • Launchpad Transactions
